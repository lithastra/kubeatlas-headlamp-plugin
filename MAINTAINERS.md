# Maintainers

The KubeAtlas Headlamp plugin shares the maintainer set of the core
project. The authoritative roster lives in the kubeatlas main
repository and is mirrored here so contributors do not have to chase
two pages:

  https://github.com/lithastra/kubeatlas/blob/main/MAINTAINERS.md

The same maintainers act as approvers for this repository. There is
no separate plugin-level governance until the plugin grows its own
sub-team.

## Becoming a maintainer

The promotion path is documented in the kubeatlas project's
governance docs. Until then, any contributor with a sustained track
record of merged changes and reviews can ask in a GitHub Discussion.
